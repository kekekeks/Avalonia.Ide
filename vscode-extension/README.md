- .NET Core runtime 2.1.

## Quickstart
- Onetime

  `git clone https://github.com/AvaloniaUI/avalonia-dotnet-templates.git ; cd avalonia-dotnet-templates ; dotnet new --install`

- When creating new project

  `dotnet new avalonia.app -o <NameOfMyApp>`
